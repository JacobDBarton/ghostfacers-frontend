# Ghostfacers

Ghostfacers is a haunted site tracker that pulls haunted site data from locations all around the United States! Get details of America's most haunted locations, search cities to find the lingering spirits that roam around hotels, prisons, asylums, or abandoned buildings. If you've been or know about the haunted location you can also leave a review!
